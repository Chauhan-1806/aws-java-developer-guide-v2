Programming with {AWS-services} using the {sdk-java}
--

This section provides programming examples you can use with the {sdkjavav2} for specific
features, use cases, and {AWS-services}.

Find the source code for these examples and others in the {AWS} documentation
{url-awsdocs-github}aws-doc-sdk-examples[code examples repository on GitHub].

To propose a new code example for the {AWS} documentation team to consider producing, create a new
request. The team is looking to produce code examples that cover broader scenarios and use cases,
versus simple code snippets that cover only individual API calls. For instructions, see the
"`Proposing new code examples`" section in the
{url-awsdocs-github}aws-doc-sdk-examples/blob/master/README.rst[Readme on GitHub].
